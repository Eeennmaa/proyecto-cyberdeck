⚠️ Zona de configuración avanzada. Estilo y lógica requeridos.

COMPONENTES

  • Raspberry Pi 4
  • Dos pantallas HDMI de 7 pulgadas
  • Teclado mecánico compacto
  • Power Bank
  • Hub USB
  • Carcasa impresa en 3D

DISEÑO Y MODELADO 3D

Se usa Software de modelado (como Fusion 360 o Blender) para crear la carcasa.

El diseño incluye:

  • Compartimientos para cada componente
  • Ranuras de ventilación.
  • Espacios para puertos accesibles.
  • Soporte para las dos pantallas en ángulo ergonómico

IMPRESIÓN Y MONTAJE

  • Se imprimen piezas en PLA o PETG
  • Se ensamblan las pantallas, teclado y Rasberry Pi dentro de la casrcasa.
  • Se asegura todo con tornillos, adhesivos y clips impresos.
  • Se conectan los cables HDMI, USB y alimentación.

CONFIGURACIÓN DEL SISTEMA

  • Se instala Rasberry Pi OS.
  • Se configuran las dos pantallas como escritorio extendido.
  • Se personaliza la interfaz con temas oscuros, íconos retro y terminales estilizadas.
  • Se instalan apps clave: navegador, terminal, editor de código, etc.
